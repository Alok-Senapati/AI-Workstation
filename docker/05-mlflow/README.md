# MLflow Layer

Provides:

- MLflow Server
- Experiment Tracking
- Model Registry

Image

```
ai-mlflow:1.0.0
```

Verify

```bash
make build
make verify
```

## Usage

Build

```bash
make build-mlflow
```

Verify

```bash
make verify-mlflow
```

Run

```bash
make up-mlflow
```

Stop

```bash
make down-mlflow
```

Logs

```bash
make logs-mlflow
```

UI

```
http://localhost:5000
```
